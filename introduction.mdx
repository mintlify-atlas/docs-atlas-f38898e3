## What is Terra?

Terra is a rolling-release Fedora repository monorepo that provides all the software you need for your Fedora system. With Terra, you can install the latest packages knowing that quality and security are assured.

This monorepo contains package manifests for hundreds of packages across multiple categories, built using the Anda package management system.

## What's in Terra?

Terra contains hundreds of packages organized into categories:

  <Card title="Applications" icon="window">

  </Card>
  <Card title="Games" icon="gamepad">

  <Card title="Development tools" icon="code">
